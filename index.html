Consilium

A movie recommender system that utilizes state of the art machine learning algorithm to learn user's preference and taste toward movies in real time.

  • The core of the artificial intelligence was trained by 22 million movie ratings from 250,000 users on 30,000 movies.
  • Recommendations are computed by collaborative filtering algorithms. The primary algorithms we used are k-Nearest Neighbors and Sparse SVD.
  • Further mathematical formulations and details can be found on Consilium's about page and on this paper .
  • Function optimizations and machine training were done with batch and stochastic gradient descents.

Heroic Bird

Object-oriented ES6 JavaScript game with custom physics engine.

Game logic and game view rendering are decoupled into separate classes. The Game class is responsible for updating states of animatable objects, while GameView class is responsible for managing the context of canvas and drawing the animatable objects in every frame.

  • HTML Canvas
  • Lots of explosion, thanks to sprite images
  • Utilized class inheritance to keep code dry
